Склонируйте репозиторий(git clone)
Для запуска проекта перейдите в папку final
Выполните команду npm/yarn  install 
Задача "css". Запускается командой "gulp css"
Задача "js". Запускается командой "gulp js"
Задача "images". Запускается командой "gulp images"
Задача "minify". Запускается командой "gulp minify"
Задача "watch". Запускается командой "gulp watch".Она следит за изменениями файлов и автоматически запускает другие задачи
Все эти задачи можно запустить командой gulp
